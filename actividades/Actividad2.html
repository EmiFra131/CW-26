Explicando un Programa en lenguaje C

Agregamos el codigo a revisar


    #include <stdio.h>

        int main() {
        int edad;
        printf("¿Cuántos años tienes? ");
        scanf("%d", &edad);

        if (edad >= 18) {
            printf("Eres mayor de edad.\n");
        } else {
            printf("Eres menor de edad.\n");
        }

        return 0;
    }

Explicacion:

#include <stdio.h>

Agregamos la biblioteca estandar de entrada y salida


int main()

Escribimos nuestra funcion principal


int edad;

Declaramos una variable de tipo entero llamado edad


printf("¿Cuántos años tienes? ");

Imprimimos en pantallla con printf la pregunta para que el usuario introduzca su edad


scanf("%d", &edad);

Con scanf pordemos redireccionar un valor que el usuario ingresa en la terminar a nuestra variable de tipo entero edad


if (edad >= 18) {
            printf("Eres mayor de edad.\n");

Con este if comprobamos si el usuario tiene mas de 18 años de edad, si es verdad se imprime en pantalla con otro printf que el usuario es mayor de edad


else {
            printf("Eres menor de edad.\n");

Si nuestra condicion anteriro es falsa, vamos a imprimir con ayuda de otro printf que el usuario es menor de edad


return 0;

Usamos return 0 para no regresar ningun valor de nuestra funcion principal y terminar el programa
